Curso Git

git init:

al iniciar un proyecto

git add:

agregar un archivo

Ejemplo: git add index.html, con esto logramos agregar el archivo

git status -s:

indica las carpetas que contiene el proyecto

Si esta en rojo '?', es porque aún no está agregado, una vez que se agrega, aparecerá '?' en color verde

git commit:

Agrega una modificación del archivo o guarda un respaldo, podemos agregar una descripción

Ejemplo: git commit -m "Agregando una modificación", antes de agregar el cambio debes primero agregar git add, sino al consultar con git status -s arrojara una 'M' en rojo, hasta no hacer git add arrojará la 'M' en verde

git config --global user.name "Your name"

Esto es para configurar GIT con tu nombre de usuario

git config --global user.email "your.email@example.com"

También debes de poner el email para que se configure de manera completa

git log --oneline:

Muestra los COMMIT que has realizado

git reset --hard (código):

Con esto, logramos volver a una versión que habíamos modificado
